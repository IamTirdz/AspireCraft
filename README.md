## Overview

**Aspire Craft** is a tool that generates **production-ready .NET backend projects** and it streamlines the setup of modern .NET backends with optional features like **.NET Aspire orchestration, authentication, caching, messaging, background processing, and testing**.  

Whether you are building a monolith or a distributed microservice, Aspire Craft helps you **scaffold the backend with best practices** in seconds.  

---

## Features

- **Project Architecture**
  - Clean Architecture  
  - Ready for multi-layered development (API, Application, Domain, Infrastructure)  

- **Framework**
  - Supports **.NET 8.0 LTS**, **.NET 9.0**, and **.NET 10.0**  

- **Authentication**
  - Identity + JWT  
  - OpenID Connect  
  - Optional authentication  

- **Database**
  - Supports **SQL Server, PostgreSQL, MySQL, and SQLite**  

- **Orchestration**
  - Optional **.NET Aspire orchestration** for distributed systems  
  - Automatically enables required features:  
    - Messaging (RabbitMQ, Kafka, or Service Bus)  
    - Distributed caching (Redis or In-Memory)  
    - Polly for resilience  

- **Caching**
  - In-memory caching for monoliths  
  - Redis for distributed scenarios  

- **Messaging**
  - RabbitMQ, Kafka, or Azure Service Bus  

- **Background Processing**
  - Hosted Services  
  - Quartz.NET  
  - Hangfire  

- **Resilience**
  - Polly integration for retries, circuit breakers, and fallback  

- **Logging**
  - Serilog integration (optional)  

- **Testing**
  - Unit Tests  
  - Integration Tests  
  - Architecture Tests
